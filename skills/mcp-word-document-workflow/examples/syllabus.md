# Example: Syllabus

## Use Case
Create a course syllabus with objectives, grading, schedule, and bibliography.

## Suggested Structure
1. Course identification
2. Description
3. Competencies/objectives
4. Units
5. Assessment table
6. Weekly schedule table
7. Bibliography
8. Policies

## Minimal Tool Flow
1. `word_create_document`
2. `word_add_heading`
3. `word_add_paragraph`
4. `word_add_table`
5. QA
