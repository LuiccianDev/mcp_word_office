# Example: Meeting Minutes

## Use Case
Document a meeting with agenda, decisions, and action items.

## Suggested Structure
1. Meeting metadata
2. Participants table
3. Agenda
4. Discussion by topic
5. Decisions
6. Action items table
7. Next meeting details

## Minimal Tool Flow
1. `word_create_document`
2. `word_add_heading`
3. `word_add_paragraph`
4. `word_add_table`
5. QA
