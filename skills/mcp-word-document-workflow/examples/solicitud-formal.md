# Example: Formal Request

## Use Case
Write an administrative or institutional request with justification and attachments.

## Suggested Structure
1. Date and recipient
2. Subject
3. Background
4. Explicit request
5. Justification
6. Closing and signature
7. Attachment list

## Minimal Tool Flow
1. `word_create_document`
2. `word_add_paragraph`
3. `word_format_text` (subject emphasis)
4. QA
