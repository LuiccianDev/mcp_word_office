# Example: Report

## Use Case
Create a formal report with summary, results, analysis, and conclusions.

## Suggested Structure
1. Cover / title
2. Executive summary
3. Context
4. Results (tables)
5. Analysis
6. Conclusions
7. Recommendations

## Minimal Tool Flow
1. `word_create_document`
2. `word_set_core_properties`
3. `word_add_heading`
4. `word_add_paragraph`
5. `word_add_table`
6. `word_format_text` / `word_format_table`
7. QA and optional `word_convert_to_pdf`
