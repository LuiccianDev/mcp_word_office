# Example: Proposal

## Use Case
Prepare a commercial or technical proposal with scope, pricing options, and timeline.

## Suggested Structure
1. Cover and proposal metadata
2. Executive summary
3. Problem statement
4. Proposed solution
5. Scope
6. Pricing/options table
7. Timeline table
8. Terms and conditions

## Minimal Tool Flow
1. `word_create_document`
2. `word_set_core_properties`
3. `word_add_heading`
4. `word_add_paragraph`
5. `word_add_table`
6. QA and optional PDF export
