# Example: University Report

## Use Case
Build an academic report with formal sections and bibliography.

## Suggested Structure
1. Institutional cover
2. Abstract
3. Introduction
4. Objectives
5. Framework / methodology
6. Development / results
7. Conclusions
8. Bibliography
9. Appendices

## Minimal Tool Flow
1. `word_create_document`
2. `word_add_heading`
3. `word_add_paragraph`
4. `word_add_table` (if needed)
5. QA
