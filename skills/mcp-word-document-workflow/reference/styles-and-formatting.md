# Styles and Formatting

Keep visual output consistent, readable, and professional.

## Default Typography

- Main font: Times New Roman (formal) or Calibri (internal)
- Body: 12pt
- H1: 14pt bold
- H2: 13pt bold
- Line spacing: 1.5 for body text

## Practical Formatting Rules

- Use `word_add_heading` for structure.
- Use `word_format_text` for local emphasis.
- Use `word_create_custom_style` only when 3+ paragraphs share the same style.
- Apply most formatting after content structure is stable.

## Common Mistakes to Avoid

- Formatting too early before content is finalized.
- Creating one-off custom styles for a single paragraph.
- Mixing many fonts without clear purpose.
- Overusing underline and colors.

## Visual Consistency Checklist

- [ ] Heading hierarchy is coherent (H1 -> H2 -> H3)
- [ ] Font choices are consistent
- [ ] Spacing is readable and uniform
- [ ] Color usage is minimal and intentional
- [ ] No unresolved placeholders remain
